# DAG Factory YAML Builder

A tool for building Apache Airflow DAGs using YAML configuration with dag-factory.

## Keywords
- airflow
- airflow-3
- dag-factory
- yaml
- data-engineering
- orchestration

## Description
This project provides utilities and examples for constructing Airflow Data Access Graphs (DAGs) using YAML-based configurations through the dag-factory library, enabling declarative workflow orchestration for data engineering pipelines.
